# Realtime Chat Web App 🚀

A modern realtime chat application with secure authentication, live messaging using Socket.IO, and persistent chat storage using MongoDB.

Built using React, Node.js, Express, MongoDB, Tailwind CSS, and Socket.IO with a clean responsive mobile-friendly UI.

---

# Features ✨

* User Signup & Login
* Realtime Messaging
* MongoDB Chat Storage
* Recent Chats Sidebar
* User Search
* Online / Offline Status
* Mobile Friendly UI
* Auto Scroll Messages
* Enter Key Message Send
* Logout System
* Chat History

---

# Installation ⚡

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/realtime-chat-app.git
```

---

# Backend Setup

```bash
cd server
npm install
```

Create `.env`

```env
PORT=5000

MONGO_URI=YOUR_MONGODB_URI

JWT_SECRET=supersecretkey
```

Run Backend

```bash
npm run dev
```

---

# Frontend Setup

```bash
cd client
npm install
npm run dev
```
